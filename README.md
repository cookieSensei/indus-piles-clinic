# Indus Piles Clinic Website

A modern **Next.js + React + TailwindCSS** website for **Indus Piles Clinic**, Kanpur.

This project is being developed as a **surprise website for Dr. Gaurav Sharma**, an Ayurvedic specialist treating Piles, Fissure, and Fistula.

**Location:** Kalyanpur, Kanpur, Uttar Pradesh

* Fistula Treatment

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/cookieSensei/indus-piles-clinic.git
```

Go to project directory

```bash
cd indus-piles-clinic
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open browser

```
http://localhost:3000
```
